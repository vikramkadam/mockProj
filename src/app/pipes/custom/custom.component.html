To create Custom Pipes angular provides Pipe and PipeTransform interface.

By default Pipes are pure , when we want to make it impure we make pure:false


1. Welcom Pipe

Show Transformed Data: {{welcomeVal | welcome}}

2. Even Odd

Is {{welcomeVal}} Even : {{welcomeVal | evenodd }}

2. Multiply by 10:

10* {{welcomeVal}} = {{welcomeVal | multiply }}

4. JSON.Stringify

JSON By nothing: {{welcomeVal}} = {{welcomeVal | seprate }}

4. kg-grams

Enter value In Kg : {{welcomeVal}} = {{welcomeVal | kgtogram }} grams.
