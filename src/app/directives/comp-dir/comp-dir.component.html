Component Directive

Component directive is used to specify the template/html for the DOM Layout

Built-In:@Component

Attribute Directive

Attribute directive is used to change/modify the behavior of the html element in the DOM Layout

Built-In:NgModel,ngStyle,ngClass

Stractural Directive

Stractural directive used to add or remove the html Element in the DOM Layout.

Built-In:*ngIf,*ngFor,*ngSwitch

Attribute Directives:

Attribute Directives:

Attribute directives manipulate the DOM by changing its behavior and appearance.
We use attribute directives to apply conditional Styles & Classes to elements, show or hide elements or dynamically change the behavior of a component according to the changng property.

Commonly used Attribute Directives:
  1. NgModel
  2. ngStyle
  3. ngClass

NgModel:

The ngModel attribute directive is used the achieve the two-way data binding.

NgModel Value is : {{ngModelValue}}

NgClass:

The ngClass is used to add ore remove the CSS classes from an HTML element.

1.This is Paragraph with normal class binding: class="red"

1.The ClassName is the property of HTML element thats why we use property binding here: [className]="'cls-green'"

1.We can add two classes with same time: [className]=" 'cls-green size' "

Conditionally apply class: [className]=" 'cls-yellow ? activatedClass : cls-red' "


1. Using ngClass we can apply classes to HTML Element: (ngClass with string) [ngClass]="'cls-red size'"

2. (ngClass with Array) [ngClass]="['cls-red size']"

3. (ngClass with Object)

We can conditionally appy class: with : *ngFor="let ele of obsData$ | async ; let flag = even" [ngClass]=" flag? 'cls-green' : 'cls-red' "

Custom Directive:

Custom Directive:

Angular CD is created to change apperance and behavior of HTML element.

  1. It is a TS class with @Directive Decorator.
  2. Use ElementRef class to directly access DOM to change host element appearance.
  3. Use @Input() decorator to accept user input in our custom directive.
  4. Use @HostListner() decorator to listen events in custom attribute directive.

We have applied custom directive style i.e color on this paragraph. ( this.eleRef.nativeElement.style = this.myStyle;)

Custom directive can also accept input from user. we have to declare @Input(). We must use property name same as selector name.

If we want to change element appearance in DOM on any event then we need to listen event in our custom directive.
To listen event we will use Angular @HostListener() decorator.
